# Password_Management
Created by [Shreyan Naskar](https://github.com/shreyan-naskar/Password_Management). It involves a basic encryption Algorithm using the Hill Cipher and relevent file handling.

## Functions :
  1. Storage and Entry of ID & Encrypted Password.
  2. Display Decrypted Password for given ID.
  3. Deleting stored record for given ID.
  4. Updating stored record for given ID.
